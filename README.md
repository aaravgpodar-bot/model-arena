# Model Arena

A prompt-first blind comparison app for judging two anonymous responses at a time.

Open `index.html` in a browser, type a prompt, and run a battle. The app generates two different built-in demo responses, hides their labels, lets you vote, then reveals which demo response you picked.

No API key is required.

## Results

The app tracks total votes, the current leading response style, response styles voted for, prompt battles, and the size of the demo model pool.
